# 📊 HR Analytics Dashboard | Power BI

## 📌 Project Overview

This project is an interactive **HR Analytics Dashboard** built using **Microsoft Power BI**. The dashboard helps HR teams monitor employee attendance, work-from-home trends, sick leave patterns, and overall workforce presence to support better decision-making.

The dashboard transforms raw attendance records into meaningful insights using Power BI visualizations, DAX measures, and data modeling.

## 🎯 Objectives

- Analyze employee attendance patterns.
- Track Work From Home (WFH) percentage.
- Monitor Sick Leave (SL) trends.
- Measure overall employee presence.
- Help HR identify attendance patterns and improve workforce planning.
